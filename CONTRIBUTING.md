# Contribuindo para o Projeto Educaboraí

O **Projeto Educaboraí** é um projeto dedicado ao suporte educacional de alunos em Itaboraí. Este documento estabelece as diretrizes para quem deseja colaborar com a plataforma, seja por meio de conteúdo pedagógico ou melhorias técnicas.

## Formas de Colaboração

### Conteúdo Acadêmico
A principal base do projeto é a transposição de matérias de cadernos escolares para o ambiente digital. As contribuições podem ser feitas das seguintes formas:
*   Fornecimento de resumos de matérias baseados em cadernos corrigidos por professores.
*   Revisão ortográfica e gramatical de conteúdos já publicados.
*   Criação de perguntas dissertativas para o sistema de fixação.

### Desenvolvimento e Usabilidade
Para colaboradores da área técnica, o foco deve ser:
*   Otimização da interface para dispositivos móveis.
*   Melhoria na lógica de validação das respostas digitadas.
*   Correção de falhas de carregamento ou navegação.

## Critérios de Qualidade

### Qualidade das Perguntas
As questões interativas devem ser exclusivamente do tipo "digitar". Ao sugerir uma pergunta, o colaborador deve fornecer também a resposta esperada e possíveis variações aceitáveis para evitar erros injustos ao aluno.

### Fidelidade ao Conteúdo Escolar
Toda matéria inserida deve ser fiel aos tópicos abordados nas escolas da região, garantindo que o aluno encontre um reflexo do que é exigido em sala de aula.

## Procedimento para Envio

1. Verifique as discussões ou tarefas abertas no repositório para evitar duplicidade de esforços.
2. Ao realizar uma alteração ou sugerir conteúdo, descreva de forma clara qual matéria ou funcionalidade está sendo afetada.
3. Certifique-se de que o texto enviado está revisado e pronto para leitura dos alunos.

## Conduta
O objetivo do projeto é puramente educacional e comunitário. Espera-se que todos os colaboradores mantenham a seriedade e o foco na utilidade pública da ferramenta.
